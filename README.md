Built a WhatsApp-based University Lecture Assistant Bot using Node.js, Meta WhatsApp Cloud API, Railway, and Google Sheets 🚀

Features:
✅ Today's lectures
✅ Tomorrow schedule
✅ Next lecture finder
✅ Subject shortcuts (ML, HPC, SSR, etc.)
✅ Auto reminders
✅ Live timetable updates from Google Sheets

Tech Stack:
Node.js | Express | Railway | WhatsApp Cloud API | Google Sheets
